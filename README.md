# Titani-Data-Analysis-Project
Titanic Data Analysis Project
Titanic Data Analysis Project

This project focuses on exploratory data analysis (EDA) and data cleaning using the Titanic dataset from Sinking of the RMS Titanic. The objective of the project was to understand raw data, identify and resolve data quality issues, and uncover insights influencing passenger survival.

Project Objectives
Perform data cleaning and preprocessing
Handle missing values and duplicates
Explore passenger demographics and travel information
Analyze factors affecting survival
Create visualizations to communicate insights effectively
Tools & Technologies
Python
Pandas
NumPy
Matplotlib
Seaborn
Jupyter Notebook
Key Insights
Female passengers had significantly higher survival rates than male passengers.
First-class passengers were more likely to survive compared to third-class passengers.
Socio-economic status and gender strongly influenced survival outcomes.
Project Deliverables
Data cleaning and preprocessing
Exploratory Data Analysis (EDA)
Data visualizations
Analytical report and insights
